# API Overview

The KoAkademy API provides a RESTful interface for developers to build applications on top of the KoAkademy platform.

## Base URL

All API requests should be prefixed with:

`https://api.koakademy.edu/api`

## Authentication

The API uses Bearer Token authentication. You can obtain a token by logging into the KoAkademy portal and generating one from your profile settings.

## Response Format

All responses are returned in JSON format. Standard HTTP status codes are used to indicate success or failure.
