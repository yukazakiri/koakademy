# Academics

The Academics section is the heart of your KoAkademy portal.

For developers, this page defines the minimum data and views that should remain available when updating enrollment, scheduling, and grading features.

## My Classes

View all the subjects you are currently enrolled in. For each class, you can see:

- Subject Code and Description
- Instructor Name
- Units / Credits
- Room Assignment

## Class Schedule

Your weekly timetable is visualized in the **Schedule** tab. It shows your classes, times, and locations (or meeting links for online classes).

## Grades

View your grades for prelims, midterms, and finals as soon as they are posted by your instructors. You can also view your Grade Point Average (GPA) for the current semester.
