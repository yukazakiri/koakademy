# Student Management API

**Everything you need to manage student data**

This API lets you list, create, update, and delete student records for KoAkademy integrations and internal tooling.

<!-- title-nav: What Can This API Do? -->

## What Can This API Do?

- 📋 List all students (with pagination)
- 👁️ View student details
- ➕ Create new students
- ✏️ Update student information
- 🗑️ Delete students

<!-- title-nav: API Overview -->

## API Overview

| Info              | Value               |
| ----------------- | ------------------- |
| **Base URL**      | `/api/students`     |
| **Auth Required** | Yes (Sanctum Token) |
| **Format**        | JSON                |

<!-- title-nav: Quick Start -->

## Quick Start

### 1. Setup Your API Client

```javascript
// JavaScript - Setup axios
import axios from "axios";

const api = axios.create({
    baseURL: "https://api.koakademy.edu/api",
    headers: {
        Authorization: "Bearer YOUR_TOKEN_HERE",
        Accept: "application/json",
    },
});
```

### 2. Make Your First Request

```javascript
// Get list of students
const response = await api.get("/students");
console.log(response.data.data);
```

<!-- title-nav: Endpoints -->

## Endpoints

### GET /api/students

**List all students with pagination**

```javascript
// Get first page of students (20 per page)
const students = await api.get("/students");

// Get specific page
const studentsPage2 = await api.get("/students?page=2");

// Get 50 students per page
const manyStudents = await api.get("/students?per_page=50");
```

### GET /api/students/{id}

**Get details of a specific student**

```javascript
// Get student by ID
const student = await api.get("/students/1");

// Or by student ID string
const student = await api.get("/students/STU-2024-001");

// Or by email
const student = await api.get("/students/john@example.com");
```

### POST /api/students

**Create a new student**

```javascript
const newStudent = await api.post("/students", {
    first_name: "John",
    last_name: "Doe",
    email: "john@example.com",
    gender: "male",
    birth_date: "2000-01-15",
    course_id: 1,
    academic_year: 1,
    student_id: "STU-2024-001",
});
```

### PUT /api/students/{id}

**Update a student**

```javascript
// Update student information
const updated = await api.put("/students/1", {
    first_name: "John",
    last_name: "Smith", // Changed from Doe
    email: "john.smith@example.com",
});
```

### DELETE /api/students/{id}

**Delete a student**

```javascript
// Delete a student (soft delete)
await api.delete("/students/1");
```

<!-- title-nav: Filtering & Sorting -->

## Filtering & Sorting

### Filter Students

```javascript
// Get only enrolled students
const enrolled = await api.get("/students?filter[status]=enrolled");

// Filter by course
const bsitStudents = await api.get("/students?filter[course_id]=1");

// Filter by year
const secondYear = await api.get("/students?filter[academic_year]=2");
```

### Sort Students

```javascript
// Sort by name (A-Z)
const sorted = await api.get("/students?sort=first_name");

// Sort by newest first
const newest = await api.get("/students?sort=-created_at");

// Sort by name (Z-A)
const sortedDesc = await api.get("/students?sort=-first_name");
```

### Select Specific Fields

```javascript
// Only get these fields (faster response)
const minimal = await api.get("/students?fields=id,first_name,last_name,email");
```

<!-- title-nav: Complete Examples -->

## Complete Examples

### JavaScript - Full CRUD

```javascript
class StudentAPI {
    constructor(token) {
        this.api = axios.create({
            baseURL: "https://api.koakademy.edu/api",
            headers: {
                Authorization: `Bearer ${token}`,
                Accept: "application/json",
            },
        });
    }

    // List all students
    async list(params = {}) {
        return this.api.get("/students", { params });
    }

    // Get one student
    async get(id) {
        return this.api.get(`/students/${id}`);
    }

    // Create student
    async create(data) {
        return this.api.post("/students", data);
    }

    // Update student
    async update(id, data) {
        return this.api.put(`/students/${id}`, data);
    }

    // Delete student
    async delete(id) {
        return this.api.delete(`/students/${id}`);
    }
}

// Usage
const api = new StudentAPI("your-token");

// List students
const { data } = await api.list({ per_page: 10 });

// Get one student
const student = await api.get("STU-2024-001");

// Create new student
const newStudent = await api.create({
    first_name: "Jane",
    last_name: "Doe",
    email: "jane@example.com",
    course_id: 1,
});

// Update student
await api.update(1, { first_name: "Jane Marie" });

// Delete student
await api.delete(1);
```

### Python Example

```python
import requests

class StudentAPI:
    def __init__(self, token):
        self.base_url = "https://api.koakademy.edu/api"
        self.headers = {
            "Authorization": f"Bearer {token}",
            "Accept": "application/json"
        }

    def list(self, **params):
        return requests.get(
            f"{self.base_url}/students",
            headers=self.headers,
            params=params
        ).json()

    def get(self, student_id):
        return requests.get(
            f"{self.base_url}/students/{student_id}",
            headers=self.headers
        ).json()

    def create(self, data):
        return requests.post(
            f"{self.base_url}/students",
            headers=self.headers,
            json=data
        ).json()

# Usage
api = StudentAPI("your-token")
students = api.list(per_page=20)
student = api.get("STU-2024-001")
new_student = api.create({
    "first_name": "John",
    "last_name": "Doe",
    "email": "john@example.com"
})
```

| ----------- | ---------------------------- |
| `enrolled`  | Currently studying           |
| `applicant` | Applied but not yet enrolled |
| `withdrawn` | Left the school              |

<!-- title-nav: Error Handling -->

## Error Handling

```javascript
try {
    const response = await api.get("/students/999");

    if (response.status === 200) {
        console.log("Student found:", response.data);
    }
} catch (error) {
    if (error.response?.status === 404) {
        console.log("Student not found");
    } else if (error.response?.status === 401) {
        console.log("Please login - token missing or invalid");
    } else {
        console.log("Something went wrong:", error.message);
    }
}
```

### Error Responses

| Status | Meaning                            |
| ------ | ---------------------------------- |
| 401    | Not logged in - check your token   |
| 403    | You don't have permission          |
| 404    | Student not found                  |
| 422    | Validation error - check your data |

<Note>Start with simple requests first. Get the list working before trying filters and sorting!</Note>

<!-- title-nav: Best Practices -->

## Best Practices

1. **Use pagination** - Don't load all students at once
2. **Select only what you need** - Use `fields=` to get faster responses
3. **Handle errors** - Always catch errors in your code
4. **Store tokens safely** - Never hardcode tokens in your code
