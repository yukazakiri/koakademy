# Quickstart Guide

Follow these simple steps to start using the portal effectively.

For contributors, this document is also the baseline expected first-run user flow in production.

## 1. Access the Portal

Navigate to `portal.koakademy.edu` in your web browser. We recommend using the latest version of Chrome, Firefox, or Edge.

## 2. Login

Enter your credentials:

- **Email**: Your official school email address.
- **Password**: The password provided by the registrar (you should change this after first login).

## 3. Set Up Profile

Go to **Profile > Settings** to:

- Upload a profile picture.
- Update contact information.
- Change your password.
- Enable Two-Factor Authentication (2FA) for extra security.
